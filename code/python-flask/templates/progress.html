Getting Started - Install the Root Certificate

To launch labs and have your browser trust the lab environment, you need to install the CyberRange root CA certificate on your machine. Select your operating system below.

  1. Download root_ca.crt using the button below.
  2. Press Win + R, type mmc, and press Enter.
  3. Go to File › Add/Remove Snap-in, select Certificates, click Add.
  4. Choose Computer account → Local Computer, click Finish then OK.
  5. Expand Certificates (Local Computer) › Trusted Root Certification Authorities › Certificates.
  6. Right-click the Certificates folder → All Tasks › Import.
  7. In the wizard, browse to the downloaded root_ca.crt, confirm the store is Trusted Root Certification Authorities, and click Finish.
  8. Click Yes on the security warning, then OK. Restart your browser.

Method 1 — Keychain Access (GUI)

  1. Download root_ca.crt using the button below.
  2. Press Cmd + Space, type Keychain Access, press Enter.
  3. Select the System keychain (applies to all users on this Mac).
  4. Go to File › Import Items and select the downloaded root_ca.crt.
  5. Double-click the imported certificate, expand the Trust section, and set When using this certificate to Always Trust.
  6. Close the window and enter your admin password when prompted. Restart your browser.
